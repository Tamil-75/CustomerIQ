# 🚀 CustomerIQ – AI Powered Customer Intelligence Platform

CustomerIQ is a Machine Learning-powered customer segmentation dashboard built using Streamlit, Plotly, Pandas, and Scikit-Learn. The platform helps businesses analyze customer behavior, identify valuable customer segments, and make data-driven marketing decisions through interactive visualizations and clustering techniques.

---

## 📌 Project Overview

CustomerIQ uses the K-Means Clustering algorithm to group customers based on their spending behavior and annual income. The dashboard provides interactive visualizations, customer insights, and cluster analysis to support business intelligence and customer analytics.

---

## ✨ Features

* 📊 Interactive Analytics Dashboard
* 🎯 Customer Segmentation using K-Means Clustering
* 📈 Age Distribution Analysis
* 💰 Income Distribution Analysis
* 🍩 Customer Segment Distribution (Donut Chart)
* 🌐 Interactive 2D Cluster Visualization
* 🚀 Interactive 3D Cluster Analysis
* 📋 Cluster-wise Business Insights
* 🎨 Premium Dark-Themed User Interface

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/CustomerIQ.git
```

Navigate to the project directory:

```bash
cd CustomerIQ
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run customer_segmentation.py
```
